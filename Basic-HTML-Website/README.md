# Basic HTML Website

A multi-page HTML website built as part of the [roadmap.sh Basic HTML Website project](https://roadmap.sh/projects/basic-html-website).

## Project Description

This project demonstrates the fundamentals of creating a structured multi-page website using semantic HTML. It includes proper meta tags for SEO, navigation between pages, and organized content sections.

## Features

- **Multi-page structure**: Home, Projects, Articles, and Contact pages
- **Semantic HTML**: Proper use of header, nav, main, section, and other semantic elements
- **SEO optimization**: Meta tags for description, keywords, and Open Graph
- **Navigation**: Consistent navigation menu across all pages
- **Content organization**: Tables, lists, and structured content layout

## Pages

- `index.html` - Homepage with work experience, education, and projects overview
- `projects.html` - Portfolio of projects
- `articles.html` - Published articles and writings
- `contact.html` - Contact information and form

## Technologies Used

- HTML5
- Semantic markup
- Meta tags for SEO
